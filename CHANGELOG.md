## Versioning

This project follows Semantic Versioning:

* **Major** version: Breaking changes or significant new functionality.
* **Minor** version: New features while maintaining compatibility.
* **Patch** version: Bug fixes and performance improvements.

Examples:

* **1.0.0** – Initial release
* **1.1.0** – Added Auto Detect and Swap Languages
* **1.2.0** – Added Voice Input
* **2.0.0** – Added OCR, AI features, and cloud synchronization
