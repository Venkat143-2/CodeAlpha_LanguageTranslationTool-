# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Elite-Language-Translator.git
```

Move into the project

```bash
cd Elite-Language-Translator
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
